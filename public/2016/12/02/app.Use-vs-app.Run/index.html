app.Run 和 app.Use 都应用于中间件, 到底有什么不同过呢?

00.00 不同之处

在 ASP.NET Core 中, 中间件是按照添加顺序执行的. 使用 app.Use 定义的中间件可能会调用下一个中间件组件.
而使用 app.Run 定义的中间件, 永远不会调用后续的中间件
